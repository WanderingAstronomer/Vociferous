# Vociferous

A modern speech-to-text dictation system with PyQt6 GUI.

## Installation

1. Create virtual environment:
   ```bash
   python -m venv .venv
   ```

2. Install dependencies:
   ```bash
   .venv/bin/pip install -r requirements.txt
   ```

3. Run:
   ```bash
   ./vociferous
   ```

Without refinement dependencies, the app runs normally but refinement features show "feature unavailable" notifications.
